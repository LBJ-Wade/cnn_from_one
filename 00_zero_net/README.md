# ZeroNet

まずは baseline として『ゼロから作るDeep Learning』の８章のＣＮＮを PyTorch で実装します。ZeroNet（勝手に命名）． 3 x 3 のフィルターを使った VGG の簡単版のようなものです。データは MNIST 数字。
